Storage Server

  • A USB drive inserted into the USB port of the Wireless Sheevaplug is exported over the network.
  • If you are running Windows on your wireless client, you can access the USB drive by going to - Run - and specifying \\192.168.1.1\
  • If you are asked for username and password, please enter 'root' as username and 'nosoup4u' as password.
  • Now you will see a shared folder called 'plug-storage'. Open it by double clicking on it.
  • You can access the data on your USB drive by clicking on either 'usb' or 'usb0' under 'plug-storage'.
  • The enire data on the USB drive will be accessible to everyone.
  • Any client that is connected to the network will be able to read and write to the USB disk.
  • Note that USB drives formatted with VFAT,EXT2 & EXT3 are supported.
