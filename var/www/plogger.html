Lamp Stack Application Host

  • The acronym LAMP refers to a solution stack of software consisting of web server(apache), MySQL(database management system) and a scripting language(Perl,Python etc.).
  • Wireless SheevaPlug can host an application which requires LAMP stack. The required packages and dependencies are included into the rootfs.
  • As an example, Wireless SheevaPlug hosts a couple of applications - an unconfigured plogger & flow player.
  • Plogger : You can access the existing plogger page at -
    this link If you haven't yet configured your plogger installation, you can do so from here
  • The configuration details are -
    host:localhost
    username:root
    password:nosoup4u
    databasename:plogger
  • Flow Player : You can access the FlowPlayer page at -
    this link
  • There you can see an example video - which is originally at the flowplayer site - being streamed in and played in your browser window.
  • Similarly, some other application requiring LAMP stack, such as Gallery (http://gallery.menalto.com) may be deployed in a similar fashion.
